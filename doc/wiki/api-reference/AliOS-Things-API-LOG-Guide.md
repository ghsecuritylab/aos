## 1-aos_set_log_level

```c
void aos_set_log_level(aos_log_level_t log_level)
```

- [x] **Description**

  Set the log level.

- [x] **Parameters**

  | IN/OUT |  NAME  |  DESC  |
  |--------|--------|--------|
  | [in] | log_level | level to be set,must be one of AOS_LL_NONE,AOS_LL_FATAL,AOS_LL_ERROR,AOS_LL_WARN,AOS_LL_INFO or AOS_LL_DEBUG. |

- [x] **Returns**

  None.
